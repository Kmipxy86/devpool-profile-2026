# DevPool Profile — แคทลียา ราโชกาญจน์ (กวาง)

หน้าเว็บแนะนำตัว 1 หน้า สำหรับสมัครเข้า **DevPool**
ตำแหน่ง พบค.5 · สังกัด ผพค.กสข.กฟต.1 · การไฟฟ้าส่วนภูมิภาค (กฟภ.)

## ทำอะไรได้บ้าง
- 🌗 สลับธีม สว่าง/มืด (จำค่าที่เลือกไว้ด้วย localStorage)
- ⚡ ปุ่มชาร์จพลัง — นับจำนวนคลิก + แถบพลังงานวิ่ง
- ⌨️ ช่องพิมพ์ข้อความ แสดงผลทันทีที่พิมพ์
- 📖 ปุ่มอ่านเพิ่มเติม/ย่อกลับ
- 🌤️ ดึงสภาพอากาศจริงจาก [Open-Meteo API](https://open-meteo.com/) ด้วย `fetch()`
- 📱 รองรับทั้งมือถือและจอใหญ่ (responsive)

## เทคโนโลยี
HTML + CSS + JavaScript ล้วน (ไม่มี framework) อยู่ในไฟล์ `index.html` ไฟล์เดียว

## วิธีเปิด
ดับเบิลคลิกที่ `index.html` เปิดด้วยเบราว์เซอร์ได้เลย

## ปรับแก้
- รายการทักษะ: แก้ในส่วน `<div class="chips">`
- เมืองในส่วนสภาพอากาศ: แก้ใน array `cities` ภายใน `<script>`
